Checklist for Outliner

Input setting and click SAVE button to submit
Please input published checklist URL.
How to add checklist on CarbonFin Outliner
  1. Open Carbonfin Outliner App
  2. Input outline as checklist
  3. Open setting
  4. Publish
  5. Copy Public link URL like as https://cfoutliner.appspot.com/?view=ZXEKN6AHX
  6. Open This Setting and paste URL above input area
  7. push SAVE button

How to add checklist on Dropbox
  1. Upload plain text list on dropbox Public folder
  2. Copy public URL of the text file like as https://www.dropbox.com/s/fmx3hsqoyracxtu/samplelist.txt?dl=0
  3. Open This Setting and paste URL above input area
  4. push SAVE button
